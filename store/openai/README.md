# 使用说明

该项目下的供应商都是兼容 OpenAI API 格式，只是部分供应商的 URL 和模型不同。


## 部署供应商

- CPA项目:[CLIProxyAPI](https://github.com/router-for-me/CLIProxyAPI)

- AIClient-2-API项目:[AIClient-2-API](https://github.com/justlovemaki/AIClient-2-API)
